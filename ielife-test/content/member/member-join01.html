  4. 兌換辦法

    華南金融集團「華得來」集點活動規範
    華南金融集團旗下華南銀行、華南永昌證券及華南產險等子公司(以下稱本集團各子公司)為回饋客戶，特別推出「華得來」集點活動(以下稱本活動)。客戶得以購買指定之金融商品或交易之點數兌換商品或服務。就本活動之具體實施規範，所有參加人均應遵守下列規定：

    參加資格

    1. 凡購買本集團各子公司指定之金融商品且簽署同意書即可參加本活動，並可使用累計之「華得來」點數兌換專屬網站所提供之商品與服務。惟本集團各子公司保留修正參加資格之權利。
    2. 前揭指定金融商品及贈點比率如下：
      1. 華南銀行：
        1. 保管箱：每箱5,000點(新承租戶年租金超過1,500元，續租戶及已享租金優惠者除外）。
        2. 黃金存摺開戶：每戶400點(完成開戶)。
        3. 黃金存摺定期定額申購：每筆200點(完成扣款)。
        4. 人身保險：每件1,500點(次月15日結算時無取消或契約撤銷之情況)。
        5. 網路銀行跨行轉帳：每筆5點(限ATM機制且該筆交易已享優惠者除外)。
        6. 申辦網路銀行：每戶20點(半年內重新申請者除外)。
        7. 開通行動網銀：每戶20點(半年內重新申請者除外)。
        8. 網路櫃檯申辦網路銀行：每戶20點(半年內重新申請者除外)。
        9. 網路櫃檯申請電子綜合對帳單：每戶20點(半年內重新申請者除外)。
        10. 網路櫃檯申請停車費代扣繳：每戶20點(同一台車半年內重新申請者除外)。
      2. 華南永昌證券：
        1. 證券交易(含權證)：每成交10,000元1點(限手續費5折以上之交易，且需申請華南永昌證券電子對帳單)。
        2. 期貨交易大台：每成交一口10點(需申請期貨電子對帳單)。
        3. 期貨交易小台(含選擇權)：每成交一口5點(需申請期貨電子對帳單)。
        4. 複委託交易：每成交約當新臺幣10,000 元1點(需申請華南永昌證券複委託電子對帳單)。
      3. 華南產險：
        1. 強制險：每件50點(保費已銷帳或兌現)。
        2. 傷害險：每件100點(保費已銷帳或兌現)。
        3. 健康險：每件100點(保費已銷帳或兌現)。
      4. 本活動之贈點對象限「自然人」客戶，保險業務之贈點對象為「要保人」。

    集點方式

    1. 客戶持有華南銀行信用卡刷卡消費所獲得之紅利點數得與「華得來」點數合併兌換，本集團各子公司保留隨時調整上述特定商品及其積點比例之相關規定之權利。
    2. 「華得來」點數僅供客戶於點數有效期間內於本網站上兌換商品或服務，客戶不得要求將點數折算現金或以其他方式給付。
    3. 客戶因取消或退還所兌換之商品或服務時，其所兌換之「華得來」點數及信用卡紅利點數將依使用比率返還，惟若點數返還時已逾有效期限者，將逕自扣除。
    4. 客戶所獲得之「華得來」點數之有效使用期間，自點數生效日起2年後年底前有效。例如：於2015年所累積之點數，最遲於2017年12月31日前使用完畢，於有效使用期間內未使用者，點數將自動失效。
    5. 客戶若對點數有爭議，須以本集團各子公司電腦資料為準。
    6. 點數於兌換商品前並不構成客戶之資產，亦不得轉讓予其他任何第三人，且客戶不得要求本集團各子公司將點數折算現金或給予其他非回饋項目之給付。

    兌換方式及標準

    1. 客戶以「華得來」點數加計華南銀行信用卡紅利點數兌換時，系統將優先扣抵「華得來」點數，不足部分始扣抵華南銀行信用卡紅利點數。
    2. 客戶得以部份點數加自付額兌換商品。自付額部份，僅能以華南銀行信用卡為支付工具。
    3. 持卡人於兌換商品，視為同意本集團各子公司及提供貨品之廠商，將其所提供之個人資料予以利用及為電腦處理發貨程序。
    4. 本集團各子公司保留兌換方式變更之權利。

    特別規定

    1. 當客戶終止與本集團各子公司之金融商品或交易之全部契約關係後，將無法參與本活動，且其剩餘的點數亦將同時失效。
    2. 所兌換之商品若在遞送過程中毀損或商品本身有瑕疵時，客戶可在收到商品七天內以電話告知本集團各子公司，請求本活動參與廠商交付無瑕疵之同種類產品，並退回原商品至發貨廠商，請務必依原廠包裝退回，如內附之說明書、保證書等。
    3. 本集團各子公司有權經公告後變更本活動之商品、參與廠商及所有兌換憑證。
    4. 客戶欲兌換之商品或服務已超出本集團各子公司所設定之限量時，本集團各子公司得以電話或書面方式公告之，並基於善意之原則，盡量協助客戶更改兌換其他商品，但客戶不得堅持兌換原欲兌換之超過限量商品。
    5. 本活動所有商品皆以實物為準，尺寸測量略有誤差，標示僅供參考，實際尺寸、顏色皆以實物為準。
    6. 本活動之商品自付額價格，可能因季節性因素、廠商專案促銷或價格調降等情形與大賣場價格有所差距，請會議依個人情況選擇兌換。
